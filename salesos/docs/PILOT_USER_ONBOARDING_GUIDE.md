## 1. Getting Started / البدء

```
Staging URL: https://staging-pilot.salesos.sa
```

> **ملاحظة:** هذا هو الرابط المؤقت لبرنامج التجربة الأولية. ستم تغييره عند الإطلاق الرسمي.


## 2. Dashboard Overview / نظرة عامة على لوحة التحكم

عند تسجيل الدخول، سترى لوحة التحكم الرئيسية التي تتضمن:

### Widget: Company Pipeline / خط أنابيب الشركات

- يعرض جميع الفرص النشطة مرتبة حسب القيمة
- ألوان المراحل: `identified` (رمادي) → `qualifying` (أزرق) → `developing` (برتقالي) → `proposing` (أخضر) → `negotiating` (ذهبي)
- اضغط على أي فرصة لعرض التفاصيل

### Widget: NBA Recommendations / توصيات الخطوة التالية

- يعرض توصيات الذكاء الاصطناعي لكل شركة
- كل توصية تحتوي على: نوع الإجراء، درجة الثقة، والسبب
- يمكنك قبول أو رفض كل توصية (انظر القسم 4)

### Widget: Signals / الإشارات

- يعرض إشارات النمو المكتشفة حديثاً
- أنواع الإشارات: `hiring` (توظيف)، `contract` (عقود)، `expansion` (توسع)، `partnership` (شراكات)، `regulation` (أنظمة)
- شدة الإشارة: `critical` (حرج) > `high` (عالي) > `medium` (متوسط)

### Widget: Tasks / المهام

- المهام المُنشأة من توصيات NBA
- المهام اليدوية المُضافة يدوياً
- الأولويات: `critical` (حرج) > `high` (عالي) > `medium` (متوسط) > `low` (منخفض)

### Widget: Timeline / الجدول الزمني

- يعرض جميع الأحداث والتفاعلات مع الشركات
- أنواع الأحداث: اجتماعات، مكالمات، إيميلات، عروض توضيحية

### Navigation / التنقل

```
لوحة التحكم (Dashboard) → العرض الرئيسي
شركات (Companies) → Company DNA وتفاصيل الشركات
فرص (Opportunities) → خط الأنابيب
توصيات (NBA) → توصيات الذكاء الاصطناعي
مهام (Tasks) → إدارة المهام
الجدول الزمني (Timeline) → سجل الأحداث
```

## 3. Company DNA / DNA الشركة

Company DNA هو العرض الشامل لكل شركة يجمع كل البيانات في مكان واحد.

### كيفية البحث عن شركة / How to Look Up a Company

1. من شريط البحث العلوي، اكتب اسم الشركة (عربي أو إنجليزي)
2. أو انتقل إلى صفحة "شركات" وتصف حسب القطاع أو المدينة
3. اضغط على اسم الشركة لفتح Company DNA

### البيانات المتاحة / Available Data

| القسم | المحتوى |
|-------|---------|
| **المعلومات الأساسية** | الاسم بالعربي والإنجليزي، رقم السجل التجاري، المدينة، القطاع، عدد الموظفين |
| **إشارات النمو** | إشارات التوظيف، العقود، التوسع، الشراكات، الأنظمة |
| **صانعو القرارات** | أسماء المديرين، المناصب، مستوى النفوذ، حالة الاتصال |
| **الفرص** | قيمة الأنابيب، مرحلة الفرصة، درجة الثقة، نية الشراء |
| **العلاقة** | قوة العلاقة، تاريخ التفاعل، الأحداث الأخيرة |
| **السياق الصناعي** | معلومات عن القطاع واتجاهات السوق |

### مثال عملي / Practical Example

عند فتح Company DNA لشركة "شركة طاقة الخليج":

```
شركة طاقة الخليج — Gulf Energy Co.
────────────────────────────────────
القطاع: الطاقة | المدينة: الرياض | الموظفون: 1,200
رقم السجل التجاري: 3010000001

إشارات النمو:
  ⚠️ حرج — فوز بعقد حكومي بقيمة 85 مليون ريال
  🔴 عالي — توظيف 45 مهندس طاقة متجددة
  🟡 متوسط — التوسع في المنطقة الغربية

صانعو القرارات:
  ✅ د. يوسف العتيبي — CEO (نفوذ عالي، متصل)
  ✅ م. سعد الغامدي — CTO (نفوذ عالي، متصل)
  ❌ أ. خالد المطيري — مدير المشتريات (نفوذ متوسط، غير متصل)

الفرص النشطة:
  SAR 3,500,000 — حلول إدارة الطاقة الذكية (Proposing)
  SAR 1,200,000 — خدمات استشارية للتحول الرقمي (Qualifying)
  SAR 850,000 — حلول الأمن السيبراني (Identified)
```

## 4. NBA Recommendations / توصيات الخطوة التالية

NBA (Next Best Action) هو نظام الذكاء الاصطناعي الذي يقترح أفضل إجراء تجاري لكل شركة.

### كيف تعمل التوصيات / How Recommendations Work

1. يحلل النظام إشارات النمو وبيانات الشركة
2. يحسب درجة الثقة (confidence score) لكل إجراء محتمل
3. يعرض التوصية الأعلى ثقة معExplanation للسبب
4. يمكنك قبول أو رفض التوصية

### أنواع التوصيات / Recommendation Types

| النوع | الوصف |
|-------|-------|
| `recommend_demo` | اقتراح عرض توضيحي للمنتج |
| `recommend_call` | اقتراح مكالمة مع صانع قرار |
| `recommend_outreach` | اقتراح التواصل مع مسؤول تنفيذي |
| `recommend_proposal` | اقتراح تقديم عرض سعر |
| `recommend_campaign` | اقتراح حملة تسويقية |
| `recommend_sequence` | اقتراح سلسلة إيميلات |

### كيفية القبول أو الرفض / How to Accept or Reject

1. انتقل إلى ويدجت "توصيات NBA" في لوحة التحكم
2. سترى توصية لكل شركة مع درجة الثقة
3. اضغط **"قبول" (Accept)** لتنفيذ التوصية — سيتم إنشاء مهمة تلقائياً
4. اضغط **"رفض" (Reject)** إذا كانت التوصية غير مناسبة — يُرجى إدخال سبب الرفض
5. يمكنك رؤية تأثير قبول/رفض التوصية على مقاييس الأداء

### دعم اللغة العربية / Arabic Language Support

- جميع التوصيات تظهر بالعربية
- أسماء الشركات بالعربية والإنجليزية
- واجهة كاملة RTL (من اليمين لليسار)
- يمكنك التبديل بين العربية والإنجليزية من الإعدادات

### مثال على توصية / Example Recommendation

```
توصية: شركة طاقة الخليج
────────────────────────────────────
الإجراء المقترح: عرض توضيحي لإدارة الطاقة الذكية
درجة الثقة: 78%
السبب: الشركة فازت بعقد حكومي كبير وتوظيف 45 مهندس، مما يشير إلى
       استعداد تقني ومالي لتطبيق حلول إدارة الطاقة الذكية.

[قبول]  [رفض]
```

## 5. FeedbackWidget / ويدجت الملاحظات

### كيفية إرسال الملاحظات / How to Submit Feedback

1. اضغط على أيقونة "ملاحظات" في الزاوية السفلية اليمنى من لوحة التحكم
2. اختر اللغة: **العربية** أو **English**
3. اكتب ملاحظاتك في صندوق النص
4. اختر تقييم NPS (انظر أدناه)
5. اضغط **"إرسال" (Submit)**

### كيفية عمل تقييم NPS / How NPS Scoring Works

**NPS (Net Promoter Score)** — صافي نقاط الترويج:

```
0-6  = مُعترض (Detractor) — غير راضٍ
7-8  = مُتريّد (Passive) — راضٍ لكن غير متحمس
9-10 = مُروّج (Promoter) — راضٍ ومحبّس
```

**السؤال:** "هل توصي بـ SalesOS لزميلك في العمل؟ (0-10)"

**كيفية حساب NPS:**
```
NPS = % المروّجين - % المعترضين

مثال:
  10 مستخدمين أجابوا
  5 أجابوا 9-10 (50% مروّجين)
  3 أجابوا 7-8 (30% متريدين)
  2 أجابوا 0-6 (20% معترضين)

  NPS = 50% - 20% = +30 (جيد)
```

### نصائح لكتابة ملاحظات فعّالة / Tips for Effective Feedback

- **اكتب بالعربية أو الإنجليزية** — كلاهما مدعوم
- **كن محدداً** — بدلاً من "الواجهة سيئة"، اكتب "لا أستطيع العثور على زر البحث في صفحة الشركات"
- **اذكر السياق** — "عند محاولة البحث عن شركة بالاسم العربي، لا تظهر النتائج"
- **اقترح حلاً** — "يفضّل أن يكون حقل البحث في أعلى الصفحة"
