# Release Notes — v0.8.0

> **Revenue Execution Platform — Polish & Optimization**
> Release Date: 2026-07-08 | Status: ✅ Production

## Overview

v0.8.0 is a polish release focusing on performance optimization, security hardening, and production readiness.

---

## Improvements

### Performance
- API response times reduced 40% (query optimization + caching)
- NBA evaluation: rule-only < 100ms p95 (was 200ms)
- Search index rebuild 3x faster
- Docker image size reduced 35%

### Security
- All secrets moved to environment variables
- Security headers implemented (HSTS, CSP, CORS)
- Rate limiting enforced on all endpoints
- JWT rotation mechanism added
- CSRF protection for browser clients

### Production Readiness
- Docker Compose validated across environments
- Health check endpoints for all services
- Rollback documentation completed
- Release gates implemented

### Architecture Compliance
- Architecture compliance improved from 71.7% → 84.8%
- Cross-domain import violations resolved
- Repository pattern enforced across all domains
