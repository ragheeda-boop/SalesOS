# Release Notes — v0.9.0

> **Wave 3: AI & Automation Platform**
> Release Date: 2026-07-11 | Status: 🟡 In Progress

---

## Overview

v0.9.0 is the **AI & Automation** release, adding RAG pipeline, workflow engine, Kafka event bus, and analytics platform.

**Version codename:** AI & Automation Platform

---

## New Features

### RAG Pipeline (Sprint 10)
- Document ingestion: PDF, DOCX, HTML, EML support
- Semantic chunking with Arabic-aware segmentation
- Self-hosted multilingual-e5-large embedding model
- pgvector vector store with HNSW index
- Hybrid search: semantic + BM25 with RRF fusion
- LLM generation with citation verification

### Workflow Automation Engine (Sprint 12)
- DAG-based workflow execution engine
- Event, schedule, and manual triggers
- 5 action types: send_email, update_crm, create_task, trigger_nba, webhook
- 5 built-in workflow templates
- Execution traces and monitoring

### Kafka Event Bus (Sprint 11)
- 3-broker Kafka cluster with KRaft mode
- Avro schema registry with versioning
- Dead Letter Queue with retry policies
- Event sourcing for critical domains
- Migration bridge from in-memory Event Runtime

### Analytics & Reporting (Sprint 13)
- Separate Data Warehouse (PostgreSQL, star schema)
- ETL pipeline with hourly incremental loads
- 5 pre-aggregated materialized cubes
- 7 standard reports + custom report builder
- Export to PDF, CSV, XLSX

### Security Enhancements
- Rate limiting on all Wave 3 endpoints
- Kafka authentication (SASL/SCRAM)
- PII stripping in document ingestion
- Read-only warehouse user

## Breaking Changes

| Change | Impact | Migration |
|--------|--------|-----------|
| Kafka replaces in-memory Event Runtime | Existing event subscribers need migration | Dual-run period of 2 weeks |
| New permission resources | `workflow:*`, `analytics:*`, `rag:*` | Review role permissions |
