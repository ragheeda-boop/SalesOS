# Release Notes — v0.7.0

> **Revenue Execution Platform — Pipeline & Intelligence**
> Release Date: 2026-07-05 | Status: ✅ Production

---

## Overview

v0.7.0 adds Pipeline Intelligence, Meeting Intelligence, and Email Intelligence.

---

## New Features

### Pipeline Intelligence (Sprint 7)
- Kanban and list views for pipeline management
- Velocity metrics: avg days per stage, cycle time
- Conversion rates per stage
- Health map: healthy / at_risk / critical
- Forecast engine: Pipeline, Best Case, Commit
- CSV and PDF export

### Meeting Intelligence (Sprint 8)
- Pre-meeting brief with company context
- Post-meeting summary with action items
- Sentiment analysis and engagement scoring
- Calendar integration (Google, Outlook)
- Activity Runtime integration

### Email Intelligence (Sprint 8)
- Sentiment analysis per email
- Topic extraction and classification
- Urgency detection
- Gmail API and Outlook API connectors
- NBA feed from email signals

---

## Improvements

- Opportunity workspace with full lifecycle management
- Deal health model with real-time indicators
- Playbook engine with assignable templates

---

## Breaking Changes

- `POST /api/v1/revenue/opportunities` now requires `company_id` (was optional)
