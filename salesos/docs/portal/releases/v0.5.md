# Release Notes — v0.5.0

> **Company Intelligence Platform**
> Release Date: 2026-06-15 | Status: ✅ Production

---

## Overview

v0.5.0 is the initial release of SalesOS — the **Company Intelligence** platform for the Saudi market.

**Version codename:** Company Intelligence Platform

---

## Features

### Company Intelligence
- Company search by name (AR/EN) and CR number
- Company profiles with branches, licenses, contacts
- Government data integration: Balady, Taqeem, Waseel
- Golden record merging with entity resolution
- Company status tracking: active, inactive, suspended, expired, liquidated

### Identity & Access
- Multi-tenant architecture
- Role-based access control (admin, manager, user, api, auditor)
- JWT authentication with refresh tokens
- API key management

### Scoring & DNA
- Company scoring engine (financial health, growth, presence, hiring)
- Company DNA profile from 50+ signals
- Knowledge graph relationships (Neo4j)
- Signal processing from government updates

### Search
- Full-text search across companies
- Filter by city, status, activity code
- Hybrid Arabic/English search

### Dashboard Widgets
- Company list view
- Company detail view
- Search widget
- Score visualization
