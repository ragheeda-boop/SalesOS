# Release Notes — v0.6.0

> **Revenue Execution Platform — NBA & Opportunity Workspace**
> Release Date: 2026-07-01 | Status: ✅ Production

---

## Overview

v0.6.0 is the **Revenue Execution** release — transforming SalesOS from intelligence to action with the NBA Engine and Opportunity Workspace.

**Version codename:** Revenue Execution Platform

---

## New Features

### NBA Engine (Sprints 5-5.0)
- 12-stage decision pipeline: Signal → Normalization → Rules → Scoring → AI → Risk → Confidence → Ranking → Output → Feedback → Learning → Audit
- 10+ business rules across stage, time, signal, health categories
- AI reasoning via GPT-4 (optional, rule-only fallback)
- 6 risk detectors: stagnation, competition, engagement, stakeholders, budget, timeline
- Confidence model with weighted scoring
- Feedback loop with weekly rule weight adjustment
- Full pipeline trace per evaluation

### Opportunity Workspace (Sprint 6)
- Full CRUD for opportunities
- Stage management with transition tracking
- NBA widget embedded in opportunity view
- Timeline widget for opportunity-specific activity
- Company snapshot integration
- Playbook assignment

---

## Breaking Changes

- New API base path: `/api/v1/revenue` for all revenue endpoints
- New permission resources: `opportunity:*`, `nba:*`, `pipeline:read`
